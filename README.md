# SignalMind AI

Cyberpunk SDR Signal Intelligence Dashboard built using:

- Next.js
- FastAPI
- Python DSP
- FFT Processing
- Recharts
- SDR Visualization

# Features

✅ Real-time waveform visualization  
✅ FFT spectrum analysis  
✅ Spectrogram rendering  
✅ Constellation / IQ visualization  
✅ Signal metrics engine  
✅ AI-based modulation estimation  
✅ Multi-page SDR dashboard  
✅ WAV signal upload pipeline  
✅ FastAPI DSP backend
